# @jy95/i18n-tools

CLI to make conversions from / to i18n files simpler.

## Export command

Export i18n files into something else.

Usage :
```bash
# Display all available exports
npx @jy95/i18n-tools export --help
```

### Export XLSX command

Usage :
```bash
npx @jy95/i18n-tools export to_xlsx [options]
```
